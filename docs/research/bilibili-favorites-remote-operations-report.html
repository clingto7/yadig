同步 Bilibili 收藏夹、收藏视频和视频-收藏夹 membership，并保留远端 resource id/type。

安全状态

  • 远端写操作不会由 LLM 自动执行，必须由用户先生成计划再确认执行。
  • move 使用确认弹窗；delete 必须输入 DELETE。
  • write session 必须具备完整 Cookie/CSRF/account identity；缺失时阻断执行。
  • 执行历史和当前预览通过共享 helper 脱敏错误，避免展示 raw cookie、CSRF、callback URL 和账号 ID。
  • blocked 错误用于登录、CSRF、captcha、rate-limit、risk-control 等需要人工处理的状态；普通 failed 可作为潜在 retryable 项展示。

已知限制

  • 当前不支持创建、重命名或删除 Bilibili 收藏夹。
  • 当前不支持 copy favorite；已实现语义是 move 和 delete。
  • 执行历史保存为新的 plan 记录，草稿和执行结果都会出现在最近历史中。
  • 真实 API 行为仍依赖 Bilibili Web 接口，可能受登录状态、风控或接口变更影响。
